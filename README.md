# Automação de Cadastro de Produtos com Python

Este projeto realiza a **automação do cadastro de produtos em um sistema web**, utilizando Python.

A automação abre o navegador, acessa o sistema, realiza o login e cadastra automaticamente vários produtos a partir de uma base de dados em CSV.

O objetivo do projeto é praticar **automação de processos e manipulação de dados**, habilidades importantes para profissionais da área de tecnologia e dados.

---

🚀 Tecnologias Utilizadas

* Python
* PyAutoGUI
* Pandas

---

📂 Estrutura do Projeto

```
automacao.py
produtos.csv
README.md
```

**automacao.py**
Script responsável por executar toda a automação.

**produtos.csv**
Base de dados contendo os produtos que serão cadastrados automaticamente no sistema.

---

⚙️ Como o Projeto Funciona

A automação executa os seguintes passos:

1. Abre o navegador Google Chrome
2. Acessa o sistema web
3. Realiza o login automaticamente
4. Importa uma base de dados de produtos
5. Preenche os campos do sistema automaticamente
6. Cadastra os produtos um por um

Tudo isso é feito utilizando automação de mouse e teclado com a biblioteca PyAutoGUI.

---

▶️ Como Executar o Projeto

1. Instale as bibliotecas necessárias:

```
pip install pyautogui pandas
```

2. Execute o script:

```
python automacao.py
```
